# Data

Raw ECG signals are **not** shipped in this repository. They are two public
12-lead corpora, downloaded from PhysioNet:

- **PTB-XL** (Germany): https://physionet.org/content/ptb-xl/
- **Ningbo / Chapman-Shaoxing** (China): https://physionet.org/content/ecg-arrhythmia/

After downloading, place the raw records under `data/raw/` (git-ignored). The
canonical loader `src/signal_loading.py` resolves paths, normalizes lead names
to the canonical order `[I, II, III, AVR, AVL, AVF, V1..V6]`, converts to
float32, and validates shape and sampling rate.

## What is shipped here (`data/processed/`)

Only **light, reproducible artifacts**:

- `metadata_combined.csv` : per-ECG fold, label, and source.
- `filter_config.json` : the frozen Butterworth 0.5-40 Hz zero-phase filter.
- `*_combined_oof.csv`, `bestmodel_oof.csv`, `bestmodel_v2_oof.csv` : canonical
  out-of-fold scores (folds 1-8) for each model.
- `*_selection*.csv` : the gate-passed feature lists per model.

## What is NOT shipped (regenerated by the notebooks)

The multi-gigabyte per-ECG **feature matrices** (`*_features.csv`) and all
caches are produced by running the extraction cells in the notebooks. They are
git-ignored and never committed.
